Ce plugin demande un antivirus installé et fonctionnel sur votre ordinateur. Lors de l'upload d'un fichier il transmet au systéme une commande que vous pouvez définir. La variable $FILE représente le chemin du fichier uploader, il faut donc l'indiquer comme cible à l'antivrus.

Les option permettent de scanné les fichiers en fonction de leur extension. Vous pouvez utiliser le joker */ pour dire que vous voulez scanner tout les fichiers. Dans ce cas la les extensions que vous mettrez après le joker ne seront pas scannées. Cette option permet d'appliquer une régle du type : Tout scanner sauf tel truc. Il est égalment possible de scanner les fichiers sans extension. Ces dernier sont accessible via l'extension : .no_ext
