Disclaimer: If you installed Pydio by the linux repository ( apt / yum ), you must install the "pydio-plugin-access.s3" adn the "pydio-plugin-metastore.s3" package before.

Use Amazon Simple Storage S3 as a file container. This extension requires the following to be installed :

  • PHP cURL extension
  • PHP OpenSSL extension
  • Amazon PHP Sdk V2 or V3 : just drop the aws.phar inside the plugin folder
Once all this is installed, you should be able to access an S3 storage, using the API keys provided on your Amazon console.
